How to create a Zapier Integration with "wise Companies"

When wise Companies and Zapier have been connected, you can create Zaps based on several different triggers, to get started:

First log in or create an account at zapier.com and click on "Create Zap" on your dashboard or Zaps list.

1. Select a Trigger

A trigger is an event that starts a Zap. Once you've set up a Zap, Zapier will monitor for that trigger event. Example. If you want to know when new user was registered in wise Companies, you can select related trigger (please see configure options bellow)

2. Select an Account

Once you select a trigger, select or create new account and provide API key, that you can receive via contact support info@wiseher.com

3. Select an Action

The next steps for setting up an action are similar to setting up a trigger. This step is telling Zapier you wish to trigger the action. Example. After user registration (emited trigger in previous step) you can send information about it to different application that available in Zapier.

Select your corresponding trigger (list described below)

That's it, you can turn on your Zap and enjoy!

List of available Trigger

1. New User

Description: Provide information after user register.

Available fields:
  • First Name: The first name of the user.
  • Last Name: The last name of the user.
  • Email: The email name of the user.
  • Country: The country name of the user.
  • Office Phone: The office phone number of the user.
  • Timezone: The timezone of the user.
  • Is Expert: Is the user an expert or not.
  • Channel Name: Name of channel, which user was connected

2. Booked Call

Description: Provide information after call was booked.

Available fields:
  • User ID: The unique identification value of the user.
  • User Name: The name of the user.
  • User Email: The email name of the user.
  • Expert ID: The unique identification value of the expert.
  • Expert Name: The name of the expert.
  • Expert Email: The email name of the expert.
  • Payment Amount: The amount of booked call.
  • Subscription Name: The name of the subscription.
